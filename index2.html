mouse

< img src = "https://upload.wikimedia.org/wikipedia/commons/thumb/c/c6/Wheel_mouse.JPG/1200px-Wheel_mouse.JPG">

El ratón o mouse (en inglés pronunciado /maʊs/) es un dispositivo apuntador utilizado para facilitar el manejo de un entorno gráfico en una computadora.
